# Deployment Logs

This directory contains deployment logs for HHDAO frontend deployments.

## Files
- `deploy-frontend-*.log` - Full deployment logs with verbose DFX output
- `replica-*.log` - Replica management and startup logs

## Troubleshooting
If deployment fails, check the most recent log files for detailed error information.

Use `tail -f logs/deploy-frontend-*.log` to monitor ongoing deployments.
