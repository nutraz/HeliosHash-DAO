# @helioshash/types

This package is a placeholder for generated ABI and type bindings (TypeChain, candid, or other).

How to use

- Put generated .d.ts/.ts files into `packages/types/src/` and run `pnpm -w build` or `pnpm --filter @helioshash/types build`.
- Consumers can import types via `import { SomeType } from '@helioshash/types'`.

Replace the placeholder `src/index.ts` with your generated index when available.
