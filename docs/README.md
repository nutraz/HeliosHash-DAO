## 🎯 National Rollout Strategy

### Phase 1: Baghpat Pilot - Micro-Scale Proof of Concept
- **Location**: Baghpat, Uttar Pradesh (early supporter advantage)
- **Scale**: 5kW solar project (scalable to 50kW)
- **Budget**: ₹60,00,000 INR starting point
- **Objective**: Prove technology, governance, and mobile app functionality
- **Outcome**: Create replicable blueprint for national scaling

### Phase 2: Urgam Valley - Macro-Scale Special Economic Zone
- **Location**: Urgam Valley, Chamoli, Uttarakhand (12-village ecosystem)
- **Vision**: Self-sustained ecological energy zone dedicated to Kalpeshwar Mahadev
- **Infrastructure**: 3MW capacity with 10km hi-tech energy corridor
- **Funding**: Global NFT participation with revenue returns

## 💰 DUAL FUNDING ARCHITECTURE

### System 1: KYC ID NFTs (Indian Users)
- **Purpose**: Identity verification and access management
- **Requirements**: Aadhaar, PAN, retina scan, mobile verification
- **Payments**: INR only via UPI/bank transfer
- **Returns**: Utility only - NO revenue sharing

### System 2: Investment NFTs (Global Participation)
- **Purpose**: Urgam Valley project funding with revenue returns
- **Access**: Open worldwide participation
- **Payments**: Crypto (ETH, BTC, stablecoins) + international fiat
- **Returns**: Revenue sharing from UrgamU operations

### Revenue Streams for Investment NFTs:
- **40%**: Cloud services from micro data centers
- **30%**: Blockchain infrastructure services
- **20%**: AI/ML computing capacity
- **10%**: Heat recapture monetization

## 🏗️ Architecture

### Core Stack
- **Frontend**: Next.js 15.5.4 (App Router) + TypeScript 5.7.3
- **UI**: shadcn/ui + Tailwind CSS (Saffron #FF9933, Green #138808, Navy #0A1A2F)
- **Blockchain**: Internet Computer (Motoko) - 9+ canisters
- **Mobile**: Responsive web-first approach + PWA capabilities

### Key Canisters
```
canisters/
├── hhdao/                 # Main DAO logic
├── identity/              # User management
├── kyc-id-nft/           # Indian user identity NFTs
├── investment-nft/       # Global participation NFTs
├── token/                # HHU/HHX token system
├── dao/                  # Governance mechanisms
└── bridge/               # Solidity ↔ Motoko bridge
```

## 🚀 Getting Started

### Prerequisites
```bash
node.js 18+
dfx 0.29.1
pnpm 10.15.1
```

### Local Development
```bash
# Clone and install
git clone [repository]
pnpm install

# Start local IC replica
dfx start --background

# Deploy canisters
dfx deploy

# Start development server
pnpm dev
```

### Mobile Testing
```bash
# Start QR code server for mobile testing
node mobile_hhdao_server.js
# Scan QR code with mobile device on same network
```
